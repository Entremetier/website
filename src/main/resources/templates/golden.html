Bootstrap und Sass

WebprojektApril 2020

Ein weiteres Projekt über eine andere imaginäre Webdesign Firma, mit dem Namen Golden, umgesetzt mit Bootstrap und Sass.

Es gibt bei diesem Projekt nur die Index Seite, auf dieser werden die einzelnen Bereiche über verschiedene id's angesteuert.

Die Bereiche und ihre Inhalte wurden ausschließlich mit Bootstrapklassen und Sass designed. Besonderheit ist die mobile Navbar, diese hat eine JavaScript function um den Content nicht im Header einzublenden sondern daneben. Darüber hinaus wurde die komplette Page nach dem mobile first Prinzip entwickelt und ist dementsprechend auch responsive.

Hier gehts zum GitHub Repository.

Golden Webdesign

Ein kurzes Video um das Programm vorzustellen und im Anschluss eine Erklärung über das Projekt

Mein erstes Projekt mit Bootstrap und Sass, es diente mir die Grundlagen dieser beiden Toolkits zu verstehen. Das vermittelte Wissen aus dem Kurs, zusammen mit dem Wissen aus dem HTML5/CSS3 Kurs haben es mir ermöglicht diese Seite zu schreiben um meine Qualifikationen, auch in diesem Bereich nachzuweisen.

Über die Seite kann mit Hilfe der Navigationsleiste gesteuert werden, dafür wurden in den einzelnen sections id's vergeben die angesteuert werden. Die komplette Seite ist dank Bootstrap responsive, nach dem Prinzip mobile first. Dabei wird die mobile Navigationsleiste über eine JavaScript function aus einem nicht sichtbaren Bereich eingeblendet. Normalerweise würde man dafür ein Plugin, zum Beispiel jQuery mmenu benutzen. Da solche Plugins kostenpflichtig sind wurde im Kurs eine andere Lösung gefunden. Ich weiße an dieser Stelle darauf hin das ich erst zum späteren Zeitpunkt einen JavaScript Kurs absolvieren werde und mich aus diesem Grund an die Vorlage des Kursleiters gehalten habe. Die function habe ich dennoch für meine Bedürfnisse angepasst, im Kurs wird die ganze Seite verschoben, in meiner Version nur das Bild im Jumbotron.

Es sind Kleinigkeiten aus den Kursen in meine Seite eingeflossen, aber es ist keine eins zu eins Übernahme. Darüber hinaus handelt es sich, bei meinem Projekt, um eine Spring Application.

Wenn Sie mehr über mein Projekt erfahren möchten können Sie hier den Quellcode in meinem GitHub Profil einsehen. Gerne können wir auch in einem persönlichem Gespräch darüber sprechen.

Allgemeines über den Kurs

ZeitraumApril bis Mai 2020

Schwerpunkt war für mich das Kennenlernen von Bootstrap und seinen Klassen als auch Sass und seiner unterschiedlichen Schreibweise im Vergleich zu CSS. In Zukunft plane ich einen JavaScript Kurs und eine weitere Spezialisierung in Bootstrap und Sass um meine eigene Webseite weiter zu verbessern und mein Portfolio auszubauen.
