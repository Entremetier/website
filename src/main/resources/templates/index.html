Sehr geehrte Damen und Herren, diese Webseite ist mein Bewerbungsschreiben an Sie. Ich möchte mich damit bei Ihnen vorstellen und Ihnen meine Erfahrungen in den verschieden Bereichen präsentieren. Im nachfolgendem Abschnitt können Sie mehr über meine Persönlichkeit und was mich ausmacht erfahren.

Diese Seite wurde von mir designed und geschrieben, alle Texte stammen ebenfalls von mir. Die Grundlagen um dieses Projekt umzusetzen habe ich in verschiedenen Kursen erworben, unter anderem Spring Framework 5, Bootstrap, Sass, HTML5 und CSS3. Über die einzelnen Kurse können Sie hier mehr erfahren. In den einzelnen Projekten erfahren Sie, darüber hinaus, mehr über meine Kurse und über ein Projekt aus dem jeweiligen Themenbereich. Des weiteren finden Sie, in der Projektübersicht, einen Link zu meinem GitHub Profil.

Sollte Ihnen meine Arbeit zusagen und Sie möchten mich kontaktieren, können Sie dafür gern dieses Kontaktformular verwenden, ich würde mich sehr über ein Feedback von Ihnen freuen.

Ich wünsche Ihnen viel Spaß mit meiner Bewerbung und freue mich schon bald von Ihnen zu hören.

Mit freundlichen Grüßen

Marco Happe

Ein Junior Java Developer aus dem schönen Wien

Mein Name ist Marco Happe und ich befinde mich zur Zeit auf der Suche nach einer neuen Herausforderung als Junior Java Developer in Wien. Im Jahr 2019 habe ich einen 6 monatigen Java Kurs am WIFI Wien absolviert und diesen mit sehr gutem Erfolg abgeschlossen. Seit September 2019 arbeite ich, zweimal wöchentlich, mit einem Privatlehrer an meinen Fähigkeiten in Java und anderen Bereichen. Des weiteren habe ich die Zeit, seit September, genutzt um mich in verschiedenen anderen Technologien einzuarbeiten. Dazu zählen unter anderem das Spring Framework, Bootstrap, Sass, MySQL, HTML5, CSS3, Git und GitHub. In näherer Zukunft plane ich einen JavaScript Kurs zu besuchen und meine Kenntnisse auf das Spring Framework zu spezialisieren.
