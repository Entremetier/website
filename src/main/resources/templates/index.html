Über mich

Ein Junior Java Developer, aus dem schönen Wien

Mein Name ist Marco Happe und ich befinde mich zur Zeit auf der Suche nach einer neuen Herausforderung als Junior Java Developer in Wien. Im Jahr 2019 habe ich einen 6 monatigen Java Kurs am WIFI Wien absolviert und diesen mit sehr gutem Erfolg abgeschlossen. Seit September 2019 arbeite ich, zweimal wöchentlich, mit einem Privatlehrer an meinen Fähigkeiten in Java und anderen Bereichen. Des weiteren habe ich die Zeit, seit September, genutzt um mich in verschiedenen anderen Technologien einzuarbeiten. Dazu zählen unter anderem das Spring Framework, Bootstrap, Sass, MySQL, HTML5, CSS3, Git und GitHub.

Über diese Seite

Diese Webseite ist mein Bewerbungsschreiben an Sie. Ich möchte mich damit bei Ihnen vorstellen und Ihnen mein aktuelles know how präsentieren.

Diese Seite wurde komplett von mir selbst designed und geschrieben, alle Texte stammen von mir persönlich. Die einzige Hilfe die ich in Anspruch genommen habe ist JavaScript, da ich meinen Kurs erst in einigen Wochen beginnen werde. Die benötigte function habe ich im Internet gesucht und dann für meine Zwecke angepasst. Die Grundlagen für diese Seite habe ich in verschiedenen Onlinekursen erworben, unter anderem Spring Framework, Bootstrap, Sass, HTML5 und CSS3. Über die einzelnen Kurse können Sie hier mehr erfahren. In den einzelnen Projekten können Sie darüber hinaus mehr über meine Kurse und natürlich über ein Projekt aus diesem Themenbereich erfahren.

Unter diesem Abschnitt finden Sie ein Bootstrap Carousel mit einigen Bildern des Quellcodes dieser Seite und einer Beschreibung welche Vorteile ich darin gesehen habe Spring, Bootstrap und Sass für diese Seite zu verwenden.

Sollten Sie mich kontaktieren wollen können Sie dafür gern das Kontaktformular am Ende dieser Seite verwenden, ich würde mich sehr über ein Feedback von Ihnen freuen.
